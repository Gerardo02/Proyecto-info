¿Que es Javascript?

JavaScript es uno de los lenguajes de programación más populares del mundo con más de veinte años de historia. También es uno de los tres principales lenguajes para desarrolladores web:

HTML: Te permite agregar contenido a una página web.

CSS: Especifica el diseño, el estilo y la alineación de las páginas web.

JavaScript: Mejora la forma en que se comportan las páginas web.

Javascript es un lenguaje estándar de programación web para la interacción y automatización de las páginas web y aplicaciones. Con java podemos hacer muchas dinámicas en el cambio de colores, caja de diálogo y validación de formularios, entre muchas otras cosas. El objetivo que fue creado es para mejorar la experiencia del usuario y la facilitación de creaciones de elementos desde la calculadora, calendario hasta aplicaciones como redes sociales.
